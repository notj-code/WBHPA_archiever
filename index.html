Download for Windows

📌사용 방법

  • 1. 프로그램 실행: 다운로드한 `gui_app.exe` 파일을 실행하세요.
  • 2. 로그인: '로그인하기' 버튼을 눌러 만방 학부모 페이지에 로그인하세요.
  • 3. 아카이빙 시작: 원하는 년도/학기를 선택한 후 '시작' 버튼을 누르세요.
  • 4. 확인: '저장된 폴더 열기'를 눌러 PDF로 저장된 통신문을 확인하세요.

🔧문제가 발생하거나 개선 사항이 있다면 GitHub 저장소를 방문해주세요.
